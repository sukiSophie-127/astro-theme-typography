### Python 代码

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 计算前 10 个斐波那契数
fib_sequence = [fibonacci(i) for i in range(10)]
print(fib_sequence)
```
