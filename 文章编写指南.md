## 创建文章

### 方法一：使用命令行工具

在项目根目录运行以下命令：

```bash
npm run theme:create
```

按照提示输入文章标题，系统会自动创建文章文件。

### 方法二：手动创建

在 `src/content/posts/` 目录下创建一个新的 `.md` 或 `.mdx` 文件。

**文件命名建议**：
- 使用中文或英文作为文件名
- 使用连字符 `-` 分隔单词
- 例如：`我的第一篇文章.md` 或 `my-first-post.md`

## Frontmatter 配置

每篇文章开头必须包含 Frontmatter 配置，使用三个连字符 `---` 包裹：

```markdown
---
title: 文章标题
pubDate: 2026-01-26
categories: ['技术']
---
```

### 必填字段

#### `title`
- **类型**：`string`
- **说明**：文章标题
- **示例**：
  ```yaml
  title: 我的第一篇文章
  ```

#### `pubDate`
- **类型**：`date`
- **说明**：文章发布日期
- **格式**：`YYYY-MM-DD` 或 `YYYY-MM-DD HH:mm:ss`
- **示例**：
  ```yaml
  pubDate: 2026-01-26
  # 或
  pubDate: 2026-01-26 14:30:00
  ```

#### `categories`
- **类型**：`string[]`
- **说明**：文章分类数组
- **可用分类**（在 `src/.config/user.ts` 中配置）：
  - 项目（proj）
  - 技术（tech）
  - 观点（view）
  - 学习（learn）
  - 生活（life）
- **示例**：
  ```yaml
  categories: ['技术']
  # 或多个分类
  categories: ['技术', '学习']
  ```

### 可选字段

#### `modDate`
- **类型**：`date`
- **说明**：文章最后修改日期
- **默认值**：无
- **示例**：
  ```yaml
  modDate: 2026-01-27
  ```

#### `description`
- **类型**：`string`
- **说明**：文章描述，用于 SEO 和文章列表摘要
- **默认值**：无
- **示例**：
  ```yaml
  description: 这是一篇关于 Astro 主题配置的文章
  ```

#### `author`
- **类型**：`string`
- **说明**：文章作者
- **默认值**：使用网站配置的作者名
- **示例**：
  ```yaml
  author: sukiSophie
  ```

#### `draft`
- **类型**：`boolean`
- **说明**：是否为草稿（草稿不会在网站上显示）
- **默认值**：`false`
- **示例**：
  ```yaml
  draft: true
  ```

#### `pin`
- **类型**：`boolean`
- **说明**：是否置顶（置顶文章会在首页优先显示）
- **默认值**：`false`
- **示例**：
  ```yaml
  pin: true
  ```

#### `banner`
- **类型**：`image`
- **说明**：文章封面图片
- **限制**：图片最大尺寸 4096 像素
- **示例**：
  ```yaml
  banner: ./cover.jpg
  ```

#### `slug`
- **类型**：`string`
- **说明**：自定义 URL 路径
- **默认值**：使用文件名
- **示例**：
  ```yaml
  slug: my-custom-url
  ```

#### `customData`
- **类型**：`string`
- **说明**：自定义数据，用于 RSS 订阅
- **默认值**：无
- **示例**：
  ```yaml
  customData: "自定义数据"
  ```

#### `commentsUrl`
- **类型**：`string`
- **说明**：自定义评论 URL
- **默认值**：无
- **示例**：
  ```yaml
  commentsUrl: "https://example.com/comments"
  ```

#### `source`
- **类型**：`object`
- **说明**：文章来源信息
- **字段**：
  - `url`: 来源链接
  - `title`: 来源标题
- **示例**：
  ```yaml
  source:
    url: "https://example.com/original"
    title: "原标题"
  ```

#### `enclosure`
- **类型**：`object`
- **说明**：附件信息（用于 RSS 订阅）
- **字段**：
  - `url`: 附件链接
  - `length`: 文件大小（字节）
  - `type`: MIME 类型
- **示例**：
  ```yaml
  enclosure:
    url: "https://example.com/podcast.mp3"
    length: 12345678
    type: "audio/mpeg"
  ```

### 完整示例

```markdown
---
title: 我的第一篇文章
pubDate: 2026-01-26
modDate: 2026-01-27
categories: ['技术', '学习']
description: 这是一篇关于 Astro 主题配置的文章
author: sukiSophie
draft: false
pin: true
banner: ./cover.jpg
slug: my-first-post
---

这里是文章内容...
```

## 图片和多媒体

### 图片存放位置

将图片文件放在以下位置：

1. **项目根目录的 `public/` 文件夹**（推荐）
   - 路径：`public/images/my-image.jpg`
   - 引用：`![图片描述](/images/my-image.jpg)`

2. **与文章同目录**
   - 路径：`src/content/posts/my-post/cover.jpg`
   - 引用：`![图片描述](./cover.jpg)`

### 图片语法

#### 基本语法

```markdown
![替代文本](图片路径)
```

#### 带标题的图片

```markdown
![替代文本](图片路径 "图片标题")
```

#### 示例

```markdown
![我的封面图](/images/cover.jpg "这是封面图")
```

### 封面图片配置

在 Frontmatter 中设置封面图片：

```markdown
---
title: 我的文章
pubDate: 2026-01-26
categories: ['技术']
banner: ./cover.jpg
---
```

封面图片会显示在：
- 文章详情页顶部
- 社交媒体分享（Open Graph）
- RSS 订阅

### 图片优化建议

- **格式**：推荐使用 WebP、JPEG 或 PNG
- **尺寸**：封面图片建议宽度 1200-1600 像素
- **文件大小**：尽量控制在 500KB 以下
- **命名**：使用英文和连字符，如 `my-image.jpg`

### 其他多媒体

#### 视频

使用 HTML 标签嵌入视频：

```html
<video width="640" height="360" controls>
  <source src="/videos/my-video.mp4" type="video/mp4">
  您的浏览器不支持视频标签。
</video>
```

#### 音频

使用 HTML 标签嵌入音频：

```html
<audio controls>
  <source src="/audio/my-audio.mp3" type="audio/mpeg">
  您的浏览器不支持音频标签。
</audio>
```

#### 外部视频（YouTube、Bilibili）

嵌入 YouTube 视频：

```html
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/视频ID"
  title="视频标题"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>
```

```html

  src="//player.bilibili.com/player.html?bvid=BV视频ID&page=1"
  scrolling="no"
  border="0"
  frameborder="no"
  framespacing="0"
  allowfullscreen="true">
</iframe>
```

## 最佳实践

```markdown
---
```

- 合理使用关键词

## 常见问题

### Q: 如何预览文章？

运行开发服务器：

```bash
npm run dev
```

然后在浏览器中访问 `http://localhost:4321/posts/文章标题/`

### Q: 如何修改文章 URL？

使用 `slug` 字段：

```markdown
---
slug: my-custom-url
---
```

### Q: 如何隐藏文章？

设置 `draft: true`：

```markdown
---
draft: true
---
```

### Q: 如何置顶文章？

设置 `pin: true`：

```markdown
---
pin: true
---
```

### Q: 如何添加多个分类？

使用数组：

```markdown
---
categories: ['技术', '学习']
---
```

### Q: 如何使用 LaTeX 公式？

确保在 `src/.config/user.ts` 中启用了 LaTeX：

```typescript
latex: {
  katex: true,
}
```

然后使用 `$` 和 `$$` 包裹公式。

### Q: 如何引用本地图片？

将图片放在 `public/` 目录或文章同目录，然后使用相对路径：

```markdown
![图片](/images/my-image.jpg)
# 或
![图片](./my-image.jpg)
```
